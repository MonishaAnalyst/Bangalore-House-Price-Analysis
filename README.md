# Bangalore House Price Analysis

## Project Overview
This project analyzes Bangalore house price data using Excel/Google Sheets.

## Tools Used
- Excel
- Google Sheets
- Pivot Tables
- Data Cleaning
- Data Visualization

## Process
1. Collected raw housing dataset
2. Cleaned and prepared the data
3. Created Pivot Tables for analysis
4. Built visualizations and dashboards
5. Generated business insights

## Key Insights
- House prices vary significantly across locations
- Property size has a strong impact on price
- Certain areas have higher average property values
- Data cleaning improved analysis accuracy
